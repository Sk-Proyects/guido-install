# Instalar Guido

Guido es el asistente de trabajo interno de VIU. Esta página tiene todo lo que necesitas
para instalarlo — solo, sin conocimientos técnicos, en unos minutos.

> Vas a necesitar tu cuenta de Google de VIU y tu cuenta de GitHub del equipo. Si no las
> tienes todavía, pídelas a tu admin antes de empezar.

## Mac

1. **[Descarga el instalador para Mac](https://github.com/Sk-Proyects/guido-install/raw/main/Instalar-Guido-Mac.zip)**
2. Abre el archivo descargado (si no se abrió solo). Vas a ver `install.command`.
3. Haz **clic derecho → Abrir** sobre `install.command` (la primera vez macOS avisa que es de
   un "desarrollador no identificado" — es esperado; con clic derecho → Abrir aparece el botón
   para continuar).
4. Sigue los pasos en pantalla: te va a pedir iniciar sesión en GitHub (se abre el navegador)
   y va a instalar todo lo demás solo.
5. Al terminar vas a tener un ícono **Guido** en tu escritorio. De ahí en adelante: doble clic
   y listo — Guido se mantiene actualizado solo.

## Windows

1. **[Descarga el instalador para Windows](https://github.com/Sk-Proyects/guido-install/raw/main/Instalar-Guido-Windows.zip)**
2. Abre el archivo descargado. Vas a ver `install.bat`.
3. Haz doble clic en `install.bat`. Si Windows muestra "Windows protegió tu PC", haz clic en
   **Más información → Ejecutar de todas formas** (es esperado — el instalador no está firmado).
4. Sigue los pasos en pantalla: inicio de sesión en GitHub (navegador) y el resto se instala solo.
5. Al terminar vas a tener **Guido** en el escritorio y en el menú inicio. Doble clic y listo.

## ¿Algo salió mal?

El instalador se puede volver a ejecutar las veces que haga falta — no rompe nada. Si te
quedas trabado, escríbele a tu admin con una foto de la pantalla.

---

*Para curiosos: estos scripts instalan las herramientas base (git, GitHub CLI, uv, Docker
guiado), descargan Guido desde el repositorio privado del equipo y dejan un lanzador que lo
actualiza automáticamente en cada apertura. El código de Guido vive en el repo privado del
equipo; acá solo está el instalador.*
